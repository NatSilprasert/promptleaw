
# PromptLaew - AI Image Generator Platform

PromptLaew is a web-based AI image generation platform that lets users generate images from viral prompts or create and share their own customized prompts with the community.

This project is the final project for the course Computer Engineering Essentials 2110221.





## Features


- Generate images from viral prompts
- Create customized prompts
- Authentication
- Save generated images
- Responsive design

## Tech Stack

- HTML, CSS, JavaScript
- Google AI API (Gemini 2.5 Flash)
- Cloudinary
- Multer
- JWT
